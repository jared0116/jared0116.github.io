搭建博客

一.Github

1.注册Github账号

搭建博客首先在Github注册一个账号:https://github.com/，点右上角的Sign up

输入信息，确认无误后按create

2.新建存储库repository

注册成功后需要建自己的储存库，按页面的右上角New repository，然后到邮箱验证，验证完之后就再一次New repository

创建成功出现下图

二.环境配置

1.安装Git

检查Git是否安装正确，输入

git –version

出现如下图就代表安装成功

2.安装Node.js

下载Node.js安装文件

下载地址：https://nodejs.org/dist/v8.9.4/node-v8.9.4-x64.msi

下载完成后（全部按NEXT就好），按下win+R，调出运行窗口

检查Node是否安装正常，输入下面俩条指令

①node -v

②npm -v

三.安装

1.安装hexo

Hexo是一款基于Node.js、快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页https://hexo.io/zh-cn?

Nodejs框架是基于V8的引擎，是目前速度最快的Javascript引擎。chrome浏览器就基于V8，同时打开20-30个网页都很流畅。Nodejs标准的web开发框架Express，可以帮助我们迅速建立web站点，比起PHP的开发效率更高，而且学习曲线更低。非常适合小型网站，个性化网站，我们自己的Geek网站！！

JS是脚本语言，脚本语言需要一个解析器才能运行。对于写在HTML页面里面的JS，浏览器充当了解析器的角色。而对于需要独立运行的JS，NodeJS就是一个解析器

2.创建工作目录

选择一个盘符创建一个文件夹hexo
